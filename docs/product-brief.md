# فرفش (Farfish) — Kuwait Activities Booking Platform

## Product Summary

A Kuwait-first activities discovery and booking app. Users browse, discover, and book fun activities in Kuwait — all in one place.

## Core Users

- End Users: Kuwaiti families, young adults, tourists
- Providers: activity businesses in Kuwait
- Admin: farfish platform operators

## Interfaces

1. User-facing app (mobile-first web app)
2. Provider dashboard (web)
3. Admin panel (web)

## Launch Scope (MVP)

- User app: discovery, activity detail, booking flow, my bookings
- Provider dashboard: auth, add listing, view bookings
- Admin panel: approve providers, view bookings
- Payments: KNET and credit card via Tap Payments
- Arabic + English support

## Stack

- React + Vite for frontends
- Node.js + Express + PostgreSQL for backend
- Tap Payments for payment integration

## Commission Model

فرفش takes 15% commission on every completed booking and deducts it from provider payout.
